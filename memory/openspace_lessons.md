## How to Use This File

This file accumulates institutional memory from open space sessions across projects.
When running `/openspace`, look here first to:
- Avoid re-exploring already-settled questions
- Build on patterns discovered in prior sessions
- Identify recurring themes worth promoting to toolbox standards

When a theme appears 2+ times across entries, consider proposing it to `/retrospective`
as a candidate for a new universal standard or skill.
