# GitHub Actions Templates

CI/CD workflow templates for GitHub Actions.

_To be populated with stack-specific workflow files as projects are built._

## Planned Templates
- `node-ci.yml` — lint, test, build for Node/TypeScript projects
- `python-ci.yml` — lint, test for Python projects
- `go-ci.yml` — vet, test, build for Go projects

## Usage
Copy the relevant workflow file to `.github/workflows/` in your project and adjust as needed.
