# TypeScript + React Standards

Standards for projects using TypeScript with React.

_To be populated via `/retrospective` after working on a TypeScript/React project, or written manually._

## Topics to Cover
- Component structure and patterns (composition over inheritance)
- State management approach (local vs. global, when to use Context vs. external store)
- Type safety conventions (strict mode, avoiding `any`)
- Testing (Vitest/Jest + React Testing Library)
- Build tooling (Vite, Next.js, etc.)
- Styling conventions (CSS modules, Tailwind, styled-components, etc.)
- File and folder organization
