# Python + FastAPI Standards

Standards for projects using Python with FastAPI.

_To be populated via `/retrospective` after working on a Python/FastAPI project, or written manually._

## Topics to Cover
- Project structure (routers, services, models, schemas)
- Dependency injection patterns
- Pydantic schema design and validation
- Database access (SQLAlchemy, async patterns)
- Testing (pytest, httpx async client, fixtures)
- Type annotations (use them everywhere)
- Environment and secrets management (pydantic-settings)
- Linting and formatting (ruff, black)
